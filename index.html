git config --global user.email ""

Aby zmienić adres e-mail przypisany do Twoich commitów w Git, możesz użyć polecenia git config --global user.email "" . Nowy adres e-mail, który ustawisz, będzie widoczny we wszystkich przyszłych commitach, które wysyłasz na GitHub.com z linii poleceń. Wszystkie zmiany dokonane przed zmianą adresu e-mail będą nadal powiązane z poprzednim adresem e-mail.
Przykład 

      $ git config --global user.email "email@example.com"
      

      $ git config --global user.email
        email@example.com
