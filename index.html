1. Komenda "git add .": dodaje zmiany z bieżącego katalogu roboczego do stage'a 2. git config --global user.name "" : zmienia nazwę, która jest powiązana z commitami w Git-cie 3. Komenda "git reset HEAD" pozwala nam przywrocic obecny 'HEAD' do okreslonego stanu z przeszlosci 4. Komenda "git log --graph --all": wyświetlanie drzewa commit'ów w konsoli w formie graph'a 5. Komenda "git commit ": służy do zapisywania zmian w określonym pliku w lokalnym repozytorium. 6. Komenda "git bisect" słuzy do znalezienia commita, który wprowadził błąd; start rozpoczyna bisecta, bad służy do oznaczenia która wersja kodu jest wersją z błędem, good do oznaczenia, która wersja nie ma błędu, a reset do zakończenie bisecta
