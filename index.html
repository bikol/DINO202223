1. Komenda "git add .": dodaje zmiany z bieżącego katalogu roboczego do stage'a 2. git config --global user.name "" : zmienia nazwę, która jest powiązana z commitami w Git-cie 3. Komenda "git reset HEAD" pozwala nam przywrocic obecny 'HEAD' do okreslonego stanu z przeszlosci 4. Komenda "git log --graph --all": wyświetlanie drzewa commit'ów w konsoli w formie graph'a
